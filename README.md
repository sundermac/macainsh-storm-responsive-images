# Responsive Images - HW #3

The MMED-1056 HW#3 showcases three different ways to create responsive images: image tag and srcset, picture element, and the background attribute in CSS. Using these methods will help optimize the user experience across all devices by avoiding bandwidth overload while still delivering high-quality images.
